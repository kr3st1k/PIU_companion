# Pump It Up Companion

* PoC of Pump it Up Info-Application like [piugame.com](https://piugame.com) made in Kotlin/Compose
* Maybe Kotlin Multiplatform?

# What does it have already?

* Pump it Up Phoenix News
* User Best Scores
* 50 Latest scores
